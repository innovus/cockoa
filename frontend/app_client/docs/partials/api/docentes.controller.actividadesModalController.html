actividadesModalController
controller in module docentes

Description

Esta es una controllador que maneja la vista en detalle de las actividades, se la utiliza cuando se le da click en ver actividades y carga una ventana modal con las actividades de un logro

Dependencies

  • $http,

    $scope, $q, $uibModalInstance, actividades, id_logro,permisoEdicion, actividadData

Methods

  • addActividad()

    Este metodo aggrega una actividad vacia para que este lista para agregar datos y agregarla en la db

  • cancel()

    Este metodo cierra el modal cuando le de en boton cancelar

  • cancelform(actividad, index)

    Este metodo se lo utiliza para que despues de dar en el boton agregar actividad si deseamos que no se guarde la actividad le damos en cancelar y se eliminara del vector de actividades

    Parameters
    ParamTypeDetails
    actividadObject

    actividad es un objeto que tiene los datos de una actividad

    indexInt

    index es la posicion en el vector actividades

  • checkPorcentaje()

    Este metodo retorna true si el porcentaje es = 100

    Returns
    Boolean

    true si el porcentaje es 100

  • deleteActividad(actividad, index)

    Este es el metodo se lo utiliza para eliminar un actividad de el vector $scope.actividades

    Parameters
    ParamTypeDetails
    actividadObject

    actividad es un objeto que tiene los datos de un actividad

    indexInt

    index es la posicion en el vector actividades de el actividad que se va a eliminar

  • ok()

    Este metodo cierra el modal cuando le de en ok y valida que todo este bien antes de hacer la transaccion

  • saveColumn(column)

    Parameters
    ParamTypeDetails
    columnObject

    column pasamos el identificador de una carga

  • updateActividad(data, actividad)

    Este es el metodo para guardar los datos de la actividad modificado temporalmente

    Parameters
    ParamTypeDetails
    dataObject

    data es la actividad con los datos que se van a actualizar

    actividadObject

    actividad es la actividad que se va a actualizar

  • validarPorcentaje(data)

    Este es el metodo para validar si un porcentaje esta entre el 100% y si se ingresan numeros

    Parameters
    ParamTypeDetails
    dataInt

    data el el porcentaje que vamos a validar

  • validarPorcentajeTotal()

    Este metodo lo reutilizamos para validar el porcentaje total
