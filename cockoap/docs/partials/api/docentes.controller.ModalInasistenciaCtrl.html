ModalInasistenciaCtrl
controller in module docentes

Description

Esta es un controlador que se lo utiliza para visualizar en una ventana modal las inasistencias de un estudiante

Dependencies

  • $scope,

    $uibModalInstance, inasistenciasEstudiante,permisoEdicion, $filter, inasistenciaData

Methods

  • cancel()

    Este metodo cierra el modal cuando le de en boton cancelar

  • ok()

    Este metodo cierra el modal cuando le de en ok

  • showEstadoInasistencia(inasistencia)

    Este metodo se lo utiliza para mostrar el estado en el select de una inasistencia

    Parameters
    ParamTypeDetails
    inasistenciaObject

    inasistencia es un objeto que contiene todos los datos de una inasistencia

  • updateEstado(inasistencia)

    Este metodo se lo utiliza para actualizar el estado de una inasistencia

    Parameters
    ParamTypeDetails
    inasistenciaObject

    inasistencia es un objeto que contiene todos los datos de una inasistencia
